# amazon-color-counter
Get information about which colors people prefer for a given product based on reviews.

### Usage
```
./amazon-color-counter.py <amazon_product_page_url>
```

### Example Output
```
+-------------+-------+
|    Color    | Count |
+-------------+-------+
|    Black    |   36  |
|  Sand/Stone |   6   |
|     Teal    |   5   |
| Purple/Gray |   3   |
|   Crimson   |   2   |
+-------------+-------+
```
